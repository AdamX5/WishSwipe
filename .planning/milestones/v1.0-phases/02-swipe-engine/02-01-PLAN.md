Run: `npm install @use-gesture/react@10.3.1 @react-spring/web@10.0.3`

Then create `convex/swipes.ts` with two mutations:

**recordSwipe** mutation:
- Args: `productId: v.id('products')`, `direction: v.union(v.literal('right'), v.literal('left'))`, `productSnapshot: v.object({ title, imageUrl, priceAmount, priceCurrency, affiliateUrl, sourceStore })` — all fields match schema exactly
- Handler: authenticate via `ctx.auth.getUserIdentity()`, look up user via `by_token` index
- Always insert to `swipes` table: `{ userId, productId, direction, swipedAt: Date.now(), productSnapshot }`
- If `direction === 'right'`: also insert to `wishlists` table: `{ userId, productId, savedAt: Date.now(), productSnapshot }`
- Returns the swipe document `_id`

**undoSwipe** mutation:
- Args: none (operates on authenticated user's most recent swipe)
- Handler: authenticate, look up user
- Query `swipes` using `by_user_time` index with `q.eq('userId', user._id)`, order `desc`, `.first()` — gets most recent swipe
- If no swipe found: return null
- `ctx.db.delete(lastSwipe._id)`
- If `lastSwipe.direction === 'right'`: query `wishlists` using `by_user_product` index with `.eq('userId', user._id).eq('productId', lastSwipe.productId)`, `.unique()`, delete if found
- Returns the deleted `lastSwipe` document (so client can restore the card)

Then create `convex/cardQueue.ts` with one query:

**getCardQueue** query:
- Args: none
- Handler: authenticate, look up user; return `[]` if not authenticated (do NOT throw — query must be safe for unauthenticated calls during initial load)
- Collect all swipe records for this user: `ctx.db.query('swipes').withIndex('by_user', q => q.eq('userId', user._id)).collect()`
- Build `swipedIds = new Set(swiped.map(s => s.productId))`
- Collect all active products: `ctx.db.query('products').filter(q => q.eq(q.field('isActive'), true)).collect()` — safe for v1 (<100 DummyJSON products); the line immediately after the `.collect()` call MUST have the comment `// TODO: replace with paginated query when product count exceeds 1024`
- Filter out swiped, take first 20: iterate products, push to queue if `!swipedIds.has(p._id)`, break when queue reaches 20
- Returns `queue` array (Doc<'products'>[] subset)

Import style: `import { mutation, query } from './_generated/server'` and `import { v } from 'convex/values'`
